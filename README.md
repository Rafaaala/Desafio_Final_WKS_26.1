# 🚗 Sistema de Gerenciamento de Aluguel de Veículos – VelozCar

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do **Desafio Final de Dados (Fábrica de Software Workshop 26.1)**, com o objetivo de criar um sistema completo para gerenciamento de aluguel de veículos.

A proposta envolveu desde a modelagem do banco de dados até a análise de dados utilizando SQL, Python e Power BI.

---

## 🎯 Objetivo

Desenvolver um sistema capaz de:

* Gerenciar clientes e funcionários
* Controlar veículos disponíveis
* Registrar contratos de aluguel
* Controlar pagamentos
* Gerenciar manutenções

---

## 🧩 Modelagem de Dados

Foi desenvolvido:

* ✔ Modelo Conceitual (E-R)
* ✔ Modelo Lógico

O banco de dados foi estruturado com as seguintes entidades:

* Cliente
* Funcionário
* Veículo
* Contrato
* Pagamento
* Manutenção

---

## 💾 Banco de Dados (MySQL)

### ✔ DDL

Criação do banco e das tabelas com:

* Chaves primárias e estrangeiras
* Restrições (NOT NULL, UNIQUE, ENUM)

### ✔ DML

* Inserção de 10 registros por tabela
* Atualizações relevantes

### ✔ DQL

* Consultas com agregação
* Consultas com JOIN

---

## 📊 Análise de Dados (SQL)

Foram realizadas consultas para análise de:

* Quantidade de clientes
* Status dos veículos
* Média de contratos
* Situação dos pagamentos

Além de JOINs para cruzamento de dados.

---

## 📈 Power BI

Foi desenvolvido um dashboard contendo:

* 📊 Distribuição do risco de crédito dos clientes
* 🏠 Proporção de clientes com e sem imóvel
* 📈 Relação entre renda mensal e total de dívidas

O objetivo foi transformar os dados em informações visuais para facilitar a identificação de padrões e apoiar a análise do perfil financeiro dos clientes do Banco Box.

---

## 🧠 Insights Obtidos

* Controle eficiente da frota de veículos
* Identificação de pagamentos pendentes
* Avaliação da produtividade dos funcionários
* Análise de custos com manutenção

---

## 🛠 Tecnologias Utilizadas

* MySQL
* SQL (DDL, DML, DQL)
* Power BI
* Python (análise de dados e ML)
* BRMW

---

## 🚀 Considerações Finais

Este projeto permitiu aplicar conceitos fundamentais de banco de dados, análise de dados e visualização, mostrando a importância da organização e interpretação de informações para tomada de decisões.
